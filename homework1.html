Meet Jen

Hello everyone! My name is Jen. I am a local native to San Francisco who enjoys going on hikes, crafting, and cooking. One of my favorite hiking trails that is very beginner friendly is Lands End. Most of the land is flat with slight inclines and stairs. Things I love to cook is Italian and Asian dishes. I also enjoy collecting old magazines and making collages and crafts out of them!

In terms of my career path, my goal by the end of this year is to become more advanced and knowledge in web development. My first introduction to web development started in high school. There I learned what file transfer protocol is, how important having all your files in one folder was, and what HTML and CSS was and what it could do. What amazed me about HTML and CSS is what I could create through words and text strung together. It was amazing and it led me to come back to web development years later which is now into CCSF's CNIT Web Development classes. Through this program, I hope to gain more skills and be able to build more complex and userfriendly websites for others and companies. One of my favorite websites to help me learn web development is freeCodeCamp. freeCodeCamp is one of my favorite websites because it provides anyone interested in learning to code free education and courses to follow. The website has courses to gain free certifications to responsive web design, JavaScript, data visualization, and more. This website has definitely influenced me to gain more interest in web development as I went through some of their responsive web design course, I was able to build new projects and webpages. It has taught me to debug, learn to customize webpages, and more.

To become a good web developer, you not only should know how to code, but you should understand the history behind web development. The Turn of the Century discusses a brief history of web development. The article discusses the old web, what the early 2000-2001 of the web looked like which was the turn of the century, and about web interface and internet technology.

"Good Links"

  • W3Schools is a place that all web developers will need. It is a go to learn to build web pages, learn languages, and have tutorials on how code works and functions. It is a good site for clarification of HTML or CSS or learning new tricks on how to build different aspects of a webpage.
  • Stack Overflow is a webpage full of forums for developers, programmers, and anyone who codes to ask questions and answer them. Even as a beginner web developer myself, I ask questions and often find answers to my questions from others who respond to other forums on Stack Overflow.
  • CSS-Tricks is a great resource for web developers. The webpage provides tricks, methods, tips, and tricks to use CSS. There are also many blog posts that can be found that relates to how CSS is being used currently and what trends in web development there are.
