Lists Usage
Unordered List Unordered list are made up of list tags. To start unordered lists, you must first define it with unordered list tags <ul> with list tags <li> and </li>, and closes the tag with </ul>. List tags are organized with bullet points by default are listed in no particular order; it is great to use this to list grocery items, menu items, and etc.
Ordered List Ordered list are started and defined with open and closed ol tags <ol>and </ol>. In between the ol tags are list tags <li> </li>. Ordered lists are used to list steps, instructions, or things you want in order. At default, ordered lists are listed with a number and used to present information in a specific order.
Definition List Defintion lists are unique as they come with two tags, one being and starts with the defined term tag <dt></dt> and the definition description tag <dd> </dd>. Definition lists are used to create glossaries, vocabulary, questions and answers, and more.
HTML
HTML stands for Hyper Text Markup Language, it is not a programming language and HTML is the meat or content shown on a webpage such as the words and sentences, special formats with headings, structured with lists and paragraphs.
CSS
CSS stands for Cascading Style Sheet, it is also not a programming lanaguage. CSS helps bring the web page to life through styling with properties and values to design how the web page should look. With CSS, developers can customize and add color, create alignment for web content, adjust size of things, create dotted, solid, and dashed colorful borders and more.
FTP
FTP stands for File Transfer Protocol. It is the internet protocol that transfer files between computers connected to the internet. A few FTP servies are FileZilla FTP Client, FUGU, and Cyberduck.
SSH
SSH also known as Secure Shell, is a network communication protocol that allows two comptuer to communciate and share date. A transfer services is SSH File Transfer Client.
